
  # Nature Hiking Navigation App

  This repository now runs as an Expo React Native app. The original UI concept came from the linked Figma design:
  https://www.figma.com/design/dx3UycBUx6nW4GRO6HBJLD/Nature-Hiking-Navigation-App

  ## Running the app

  Install dependencies with `pnpm install`.

  Start the Expo dev server with `pnpm start`.

  Use `pnpm android` or `pnpm ios` for a target-specific launch.
